# quiet-money-images

Public image host for Quiet Money rendered post images.

Each file is `<item-id>.png`, referenced by the posting pipeline via
`https://raw.githubusercontent.com/anmoljain123/quiet-money-images/main/<item-id>.png`
for the Instagram/Facebook Graph API, which requires a publicly reachable image URL.

Managed automatically by `lib/upload_image.py` in the quiet-money repo — nothing here
is hand-edited.
